創夾資訊一群熱愛著資訊且平均年齡不到30 歲的年輕人所創立的資訊工作室，我們於 2016 年晚秋正式成立，正如同工作室名稱我們認為專案即為一個資料夾的概念，我們以新穎的架構以及紮實的技術豐富這個資料夾，最後將此資料夾封裝並且交給客戶。每個專案我們均採用 Git 進行版本控制保障每個研發人員皆可以快速的上手進行研發，並且採用 Travis CI 以及 Docker 以保障那些容易被其他團隊忽略的單元測試以及部屬建置整合的部分。


Python / JavaScript / C#


秉持著與時俱進並且以技術作為研發的優先考量，採用著集結社群力量的開放原始碼的框架進行 Web / App 的研發，並且提供完善的技術支援以及售後維運。我們擅長的程式語言以 Python , JavaScript 以及 C# 為主，包括著 Flask, Vue.js, React 以及 Xamarin 等主流的全端以及行動技術的掌握，以成熟的 MVC 架構搭配新形態的 NoSQL 資料庫例如 MongoDB / DocumentDB 進行全端之實作，且我們在伺服器虛擬化例如 ESXi , XEN 以及 OpenStack 的規劃部屬已經有多年的經驗，主流的 AWS , Azure 以及 Bluemix 雲平臺部屬也駕輕就熟。
